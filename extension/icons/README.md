# 图标文件说明

此目录需要包含以下图标文件：

- `icon16.png` - 16x16 像素
- `icon48.png` - 48x48 像素
- `icon128.png` - 128x128 像素

## 临时解决方案

在开发阶段，如果没有图标文件，可以：

1. 使用在线工具生成简单图标：
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/

2. 或者使用任何图片编辑工具创建简单的纯色图标

3. 将生成的图标文件放在此目录下

## 图标设计建议

- 使用简洁的设计
- 主色调可以使用黄色（与高亮颜色一致）
- 可以使用字母 "C" 或电影胶片图标
- 确保在小尺寸下清晰可见
